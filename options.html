Welcome to Salesforce.com Usage Enhance Extension

The aim of this chrome extension is to provide some tools and simple
interface changes to those users who need them.

This is a list of features that you will be able to use thanks to
this extension.
  • Inteface Enhancements
    • Schema Builder enhancements
      • Hide Salesforce header
      • Hide All fields
      • Unwrap All fields
  • Login Helper
    • Store credentials and tokens
    • Login with a single click from login screen

Login Helper

With Login Helper you will be able to store Salesforce.com credentials,
so you don't have to worry about having them in your mind every time
you want to get into salesforce.

Not only allows you to store credentials grouped by Project and ORG
but it can also store Tokens.

This means you will have everything login-related in the same place.
